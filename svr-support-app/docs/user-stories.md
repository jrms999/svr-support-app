# User Stories Backlog

## Resident
- As a resident, I want to log in securely so my information is protected.
- As a resident, I want to see my utility balance so I know where I stand.
- As a resident, I want to pay utilities easily to avoid arrears.
- As a resident, I want clear health and crisis contacts when I need help.
- As a resident, I want local support info based on my SVR site.

## Staff (Phase 2)
- As staff, I want to post notices so residents are informed.
- As staff, I want to update utility balances securely.

## Accessibility
- As a user, I want large buttons and clear text so the app is easy to use.
